//What is Shipcrawler

Shipcrawler is a maritime OSINT reconnaissance tool developed by Oversight. It searches Shodan, AIS databases, certificate transparency logs, and public web sources to identify internet-facing systems on commercial vessels — VSAT management portals, exposed remote access, unsecured network services, and other attack surface. Its methodology is published and peer-reviewed in IEEE Access (2026).

Designed for fleet managers, not security engineers, each report delivers a plain-language breakdown of what's exposed, how an attacker could exploit it, and what to fix first — prioritised by severity.
