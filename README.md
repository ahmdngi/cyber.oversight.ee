# cyber.oversight.ee

Cybersecurity & infrastructure landing page for operational environments.

**Stack:** Vanilla HTML/CSS/JS — no frameworks.

## Features

- Digital rain (RainingLetters) background effect, confined to side gutters
- Glitch-scrambling hero title that rotates through taglines
- Hallmark design system with dark/light mode
- Zero-dependency, single-page site served via Python `http.server`

## Design

- Fonts: Inter + JetBrains Mono (Google Fonts)
- Tokens-based CSS with oklch color space
- Accent: green 160° hue
- Paper/ink contrast for readability
